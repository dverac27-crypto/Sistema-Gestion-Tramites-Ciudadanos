# Sistema de Gestión de Trámites y Quejas Ciudadanas

Sistema desarrollado para la Alcaldía de Milagro, Ecuador, como prototipo funcional para la asignatura de Introducción a la Ingeniería de Software.

## Características
* Registro de incidencias ciudadanas.
* Seguimiento en tiempo real de trámites.
* Panel administrativo para gestión de reportes.
